# NOVA AI Assistant

NOVA is a voice-controlled AI assistant inspired by JARVIS. It combines AI chat, browser automation, navigation, news updates, weather information, YouTube control, and a futuristic PyQt6 HUD interface into a single desktop assistant.

## Features

- 🎤 Wake Word Activation (Nova)
- 🤖 AI Chat Assistant
- 🌐 Browser Automation
- 📍 Google Maps Navigation
- 📰 Live World News Summaries
- 🌦️ Weather Information
- 🎵 YouTube Search & Control
- 🧠 Memory System
- 🔊 Text-to-Speech (Edge TTS)
- 🎨 Futuristic HUD Interface
- 📊 System Monitoring (CPU, RAM, Battery)
- 🔍 Product Search Integration

## Tech Stack

- Python
- PyQt6
- Selenium
- Edge TTS
- SpeechRecognition
- BeautifulSoup
- Requests
- Groq API
- HTML/CSS/JavaScript
